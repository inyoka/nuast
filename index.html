About NUAST Computer Science

NUAST Computer Science students learn to design, build, and deploy real software. This site is your hub for accessing shared code, learning resources, and collaborative projects hosted on GitHub. Whether you’re just starting out or working on an advanced project, you’ll find everything you need here.

Learn to Code

Follow along with class exercises, example code, and step-by-step tutorials hosted in our GitHub repositories.

Collaborate

Work together on group projects using GitHub's pull requests, issues, and code reviews — just like professional developers.

Showcase Work

Publish your finished projects as GitHub Pages sites and share them with classmates, teachers, and future employers.
